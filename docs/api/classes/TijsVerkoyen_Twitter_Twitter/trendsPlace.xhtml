TijsVerkoyen\Twitter\Twitter::trendsPlace

Returns the top 10 trending topics for a specific WOEID, if trending information is available for it.

The response is an array of "trend" objects that encode the name of the trending topic, the query parameter that can be used to search for the topic on Twitter Search, and the Twitter Search URL. This information is cached for 5 minutes. Requesting more frequently than that will not return any more data, and will count against your rate limit usage.

    Signature

    public function trendsPlace(string $id, [string[optional] $exclude = null] )

    Parameters

    $id — string
    The Yahoo! Where On Earth ID of the location to return trending information for. Global information is available by using 1 as the WOEID.
    $exclude — object
    Setting this equal to hashtags will remove all hashtags from the trends list.


    Returns

    array
