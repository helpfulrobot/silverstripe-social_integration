TijsVerkoyen\Twitter\Twitter::statusesRetweetsOfMe

Returns the most recent tweets authored by the authenticating user that have recently been retweeted by others. This timeline is a subset of the user's GET statuses/user_timeline.

    Signature

    public function statusesRetweetsOfMe([int[optional] $count = null, [string[optional] $sinceId = null, [string[optional] $maxId = null, [bool[optional] $trimUser = null, [bool[optional] $includeEntities = null, [bool[optional] $includeUserEntities = null]]]]]] )

    Parameters

    $count — object
    Specifies the number of records to retrieve. Must be less than or equal to 100. If omitted, 20 will be assumed.


    $sinceId — object
    Returns results with an ID greater than (that is, more recent than) the specified ID. There are limits to the number of Tweets which can be accessed through the API. If the limit of Tweets has occured since the since_id, the since_id will be forced to the oldest ID available.


    $maxId — object
    Returns results with an ID less than (that is, older than) or equal to the specified ID.


    $trimUser — object
    When set to true, each tweet returned in a timeline will include a user object including only the status authors numerical ID. Omit this parameter to receive the complete user object.


    $includeEntities — object
    The tweet entities node will be disincluded when set to false.


    $includeUserEntities — object
    The user entities node will be disincluded when set to false.


    Returns

    array
