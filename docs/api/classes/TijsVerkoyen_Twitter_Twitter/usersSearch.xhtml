TijsVerkoyen\Twitter\Twitter::usersSearch

Run a search for users similar to the Find People button on Twitter.com; the same results returned by people search on Twitter.com will be returned by using this API.

Usage note: It is only possible to retrieve the first 1000 matches from this API.

    Signature

    public function usersSearch(string $q, [int[optional] $page = null, [int[optional] $count = null, [bool[optional] $includeEntities = null]]] )

    Parameters

    $q — string
    The search query to run against people search.
    $page — object
    Specifies the page of results to retrieve.


    $count — object
    The number of potential user results to retrieve per page. This value has a maximum of 20.


    $includeEntities — object
    The entities node will be disincluded from embedded tweet objects when set to false.


    Returns

    array
