TijsVerkoyen\Twitter\Twitter::blocksDestroy

Un-blocks the user specified in the ID parameter for the authenticating user. Returns the un-blocked user in the requested format when successful. If relationships existed before the block was instated, they will not be restored.

    Signature

    public function blocksDestroy([string[optional] $userId = null, [string[optional] $screenName = null, [bool[optional] $includeEntities = null, [bool[optional] $skipStatus = null]]]] )

    Parameters

    $userId — object
    The ID of the potentially blocked user. Helpful for disambiguating when a valid user ID is also a valid screen name.


    $screenName — object
    The screen name of the potentially blocked user. Helpful for disambiguating when a valid screen name is also a user ID.


    $includeEntities — object
    The entities node will not be included when set to false.


    $skipStatus — object
    When set to either true, t or 1 statuses will not be included in the returned user objects.


    Returns

    array
