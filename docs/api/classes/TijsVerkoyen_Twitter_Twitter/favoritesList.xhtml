TijsVerkoyen\Twitter\Twitter::favoritesList

Returns the 20 most recent Tweets favorited by the authenticating or specified user.

    Signature

    public function favoritesList([string[otpional] $userId = null, [string[otpional] $screenName = null, [int[optional] $count = 20, [string[otpional] $sinceId = null, [string[otpional] $maxId = null, [bool[optional] $includeEntities = null]]]]]] )

    Parameters

    $userId — object
    The ID of the user for whom to return results for.


    $screenName — object
    The screen name of the user for whom to return results for.


    $count — integer
    Specifies the number of records to retrieve. Must be less than or equal to 200. Defaults to 20.


    $sinceId — object
    Returns results with an ID greater than (that is, more recent than) the specified ID. There are limits to the number of Tweets which can be accessed through the API. If the limit of Tweets has occured since the since_id, the since_id will be forced to the oldest ID available.


    $maxId — object
    Returns results with an ID less than (that is, older than) or equal to the specified ID.


    $includeEntities — object
    The entities node will be omitted when set to false.


    Returns

    array
