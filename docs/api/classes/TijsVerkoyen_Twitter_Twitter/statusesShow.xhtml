TijsVerkoyen\Twitter\Twitter::statusesShow

Returns a single Tweet, specified by the id parameter. The Tweet's author will also be embedded within the tweet.

    Signature

    public function statusesShow(string $id, [bool[optional] $trimUser = null, [bool[optional] $includeMyRetweet = null, [bool[optional] $includeEntities = null]]] )

    Parameters

    $id — string
    The numerical ID of the desired Tweet.
    $trimUser — object
    When set to true, each tweet returned in a timeline will include a user object including only the status authors numerical ID. Omit this parameter to receive the complete user object.


    $includeMyRetweet — object
    When set to true, any Tweets returned that have been retweeted by the authenticating user will include an additional current_user_retweet node, containing the ID of the source status for the retweet.


    $includeEntities — object
    The entities node will be disincluded when set to false.


    Returns

    array
